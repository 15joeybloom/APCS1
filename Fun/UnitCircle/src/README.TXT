USER INSTRUCTIONS:
This program quizzes the user on trig functions 
of the unit circle, so from -2pi to 2pi. The user
is prompted with a trig function of an angle
and should type in the answer as fast as
possible. type square roots as rt, no parentheses
needed. Example: rt3/2 is half of the square root 
of 3.
